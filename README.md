# Shroomlight Studio Website

This is the official website for our creative studio, showcasing experimental games, web projects, and custom tools developed in-house. The site is built with Next.js, fully optimized for static pages and serverless Edge functions, and deployed on Cloudflare Pages.

## Features
- Games & Projects Showcase – Display experimental indie games and creative projects.
- Custom Engine Section – Highlights the Shroom Engine and its capabilities.
- Web & Experiments – Interactive coding projects and generative art demos.
- Creative Tools – Utilities and asset generators for developers and creators.
- Notify API – Users can sign up for updates via an Edge-compatible /api/notify route.
- Responsive Design – Fully mobile-friendly and optimized for all screen sizes.
- Smooth Animations – Motion effects powered by Framer Motion.

## Tech Stack
- Frontend: Next.js 16.x (with Turbopack)
- Styling: Tailwind CSS
- Animations: Framer Motion
- Serverless / Edge Functions: Cloudflare Pages Edge Runtime
- Email Notifications: SendGrid API
